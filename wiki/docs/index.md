# EnthusiaMarket

EnthusiaMarket is the stall, shop, and auction system powering the player economy on [EnthusiaSMP](https://enthusiasmp.com). Pick where you want to go:

- **[Getting Started](getting-started/welcome.md)** — New to the market? Start here.
- **[Players](players/how-do-i.md)** — How everything works, organized by feature.
- **[Admins](admins/installation.md)** — Install, configure, and manage.

In-game, type `/shop` for the player menu or `/em` for admin commands.
