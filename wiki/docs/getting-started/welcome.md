# Welcome to EnthusiaSMP

This page orients you to the server and the market system in about two minutes. When you're ready for the hands-on tour, head to **[Your first stall](walkthrough.md)**.

## What is EnthusiaSMP?

EnthusiaSMP is a semi-anarchy survival multiplayer server where anything goes — except the systems. Build, trade, and compete in a true sandbox where only your reputation is safe. Owned and operated by FainNoir with plugin development by BadgersMC.

## What does EnthusiaMarket add?

EnthusiaMarket is the server's marketplace plugin. It lets you:

- **Own a stall** — buy or bid on a protected WorldGuard region and make it yours.
- **Run a shop** — place a sign on a container and sell items, buy items from players, or trade item-for-item.
- **Search for goods** — use `/shop search <item>` to find shops selling what you need.
- **Trade with barter** — set up item-for-item trades with shop vaults for collecting payments.
- **Earn guild income** — guilds can own stalls, run markets, and set trade tariffs on other guilds.
